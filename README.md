# ToursVan Homepage

A vanilla HTML / CSS / JavaScript redesign of the ToursVan homepage. DoorDash interaction shape, ToursVan brand blue.

## Stack

- Vanilla HTML + CSS + JavaScript
- No framework, no build step, no npm
- Font Awesome 6 Free via CDN
- Google Fonts (Inter) via CDN


## Preview

Open `index.html` directly in any modern browser, or serve locally:

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000

# OR with VS Code Live Server extension
```
